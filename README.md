# HackVerse 2026 — Hackathon Website

This project is the official website for HackVerse 2026, a school/university-level hackathon.  
The site includes the event details, tracks, timeline, prizes, FAQs, registration link, and a live countdown timer.  
Design is fully responsive and built using plain HTML, CSS, and JavaScript.

## Features
- Countdown timer for the hackathon date
- Animated starfield background
- Event timeline with important dates
- Tracks and prize details
- Registration button linked to Google Form
- FAQ section with expandable questions
- Stats that count up when scrolling into view

## Tech Used
- HTML  
- CSS (Grid, Flexbox, animations)  
- JavaScript (canvas animation, counters)

## Folder Structure
HackVerse/
│── Index.html
│── style.css
│── script.js

## How to Run
Just open `Index.html` in your browser.  
No backend or setup needed.
